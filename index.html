Minha trajetória até aqui

Porém decidi ir para o desenvolvimento web utilizando linguagem javascript, na qual sigo estudando e desenvolvendo pequenos projetos pessoais, fazendo front-end com ReactJs e back-end utilizando a linguagem NodeJs.
